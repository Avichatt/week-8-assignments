# Week 8 Thursday: Time Series and Sequential Data

## Overview
This repository contains the end-to-end `assignment.ipynb` addressing both stock future price prediction and chat-log customer churn prediction.

## Requirements
* **Python Version**: 3.10+
* **Dependencies**: 
    - `pandas`
    - `numpy`
    - `scikit-learn`
    - `torch`
    - `matplotlib`
    - `jupyter`

## Data Requirements
**IMPORTANT:** The TA validation script runs the notebook from a clean environment. Ensure `stock_prices.csv` and `chat_logs.csv` are located in `./data/` before executing the notebook. A helper script `prep_data.py` has been included if you wish to synthetically generate them from the Kaggle dataset.

## Execution
Run the following in the command line from this directory:
```bash
jupyter notebook assignment.ipynb
```
Select "Run All".
